# finance-manager

A Java Spring Boot Finance Manager application for tracking user finances.

## Overview

- Built with Spring Boot, Maven
- Includes basic user entity, service and repository layers

## Getting started

1. Install Java 17+ and Maven
2. Build: `mvnw package`
3. Run: `mvnw spring-boot:run`
